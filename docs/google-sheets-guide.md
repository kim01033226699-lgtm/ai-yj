# 구글 시트 입력 가이드

현재 관리자 페이지에 설정한 프리셋 답변과 카테고리 데이터를 구글 시트에 입력하는 방법입니다.

## 1. 프리셋 답변 시트

### 시트 구조

첫 번째 행(헤더):
```
카테고리 | 레벨1_ID | 레벨1_라벨 | 레벨2_ID | 레벨2_라벨 | 답변
```

### 입력 규칙

1. **카테고리**: `support`, `campus`, `appointment` 또는 커스텀 카테고리 ID
2. **레벨1_ID**: 최상위 옵션의 ID (예: `A`, `B`, `C`)
3. **레벨1_라벨**: 최상위 옵션의 표시 이름 (예: `A 옵션`, `B 옵션`)
4. **레벨2_ID**: 하위 옵션의 ID (예: `A-가`, `A-나`, `B-가`)
5. **레벨2_라벨**: 하위 옵션의 표시 이름 (예: `가`, `나`, `다`)
6. **답변**: 최종 답변 내용 (줄바꿈은 `\n` 또는 실제 줄바꿈 사용 가능)

#### 레벨1에 직접 답변이 있는 경우
레벨2가 없고 레벨1에 직접 답변이 있는 경우:

| 카테고리 | 레벨1_ID | 레벨1_라벨 | 레벨2_ID | 레벨2_라벨 | 답변 |
|---------|---------|-----------|---------|-----------|------|
| support | D | D 옵션 | | | D 옵션에 대한 직접 답변입니다. |

레벨2_ID와 레벨2_라벨은 비워두고, 답변만 입력합니다.

## 2. 카테고리 시트

### 시트 구조

첫 번째 행(헤더):
```
id | label | emoji | description
```

### 입력 규칙

1. **id**: 카테고리 고유 ID (예: `support`, `campus`, `appointment`, `custom-1234567890`)
2. **label**: 카테고리 표시 이름 (예: `지원금`, `금융캠퍼스`)
3. **emoji**: 카테고리 이모지 (예: `💰`, `🏫`, `📋`)
4. **description**: 카테고리 설명 (선택사항)

### 예시 데이터

현재 `categories.json`의 내용을 시트 형식으로 변환하면:

| id | label | emoji | description |
|----|-------|-------|-------------|
| support | 지원금 | 💰 | 지원금 관련 문의 |
| campus | 금융캠퍼스 | 🏫 | 금융캠퍼스 관련 문의 |
| appointment | 위촉 | 📋 | 위촉 관련 문의 |

### 커스텀 카테고리 추가 예시

| id | label | emoji | description |
|----|-------|-------|-------------|
| support | 지원금 | 💰 | 지원금 관련 문의 |

| appointment | 위촉 | 📋 | 위촉 관련 문의 |
| custom-1234567890 | 새 카테고리 | 📁 | 새 카테고리 설명 |

## 3. 구글 시트 설정 방법

### 단계별 가이드

1. **구글 시트 생성**
   - [Google Sheets](https://sheets.google.com) 접속
   - 새 스프레드시트 생성
   - 시트 이름: "프리셋 답변" 또는 원하는 이름

2. **프리셋 답변 시트 작성**
   - 첫 번째 행에 헤더 입력: `카테고리,레벨1_ID,레벨1_라벨,레벨2_ID,레벨2_라벨,답변`
   - 두 번째 행부터 데이터 입력 (위 예시 참고)

3. **카테고리 시트 작성** (별도 시트 또는 별도 파일)
   - 첫 번째 행에 헤더 입력: `id,label,emoji,description`
   - 두 번째 행부터 데이터 입력 (위 예시 참고)

4. **시트 공개 설정**
   - 우측 상단 "공유" 버튼 클릭
   - "링크가 있는 모든 사용자" 선택
   - 권한: "뷰어" 선택
   - "완료" 클릭

5. **시트 ID 확인**
   - URL에서 시트 ID 복사
   - 예: `https://docs.google.com/spreadsheets/d/1ABC123...XYZ/edit`
   - 시트 ID: `1ABC123...XYZ` (슬래시 사이의 긴 문자열)

6. **환경 변수 설정**
   - `.env.local` 파일에 추가:
   ```
   NEXT_PUBLIC_GOOGLE_SHEET_ID=1ABC123...XYZ
   NEXT_PUBLIC_GOOGLE_CATEGORIES_SHEET_ID=1DEF456...ABC
   ```

## 4. 주의사항

1. **쉼표(,)가 포함된 내용**: 답변에 쉼표가 있으면 따옴표로 감싸거나, 구글 시트가 자동으로 처리합니다.
2. **줄바꿈**: 답변에 줄바꿈이 필요하면 시트에서 Alt+Enter로 줄바꿈을 입력하세요.
3. **빈 셀**: 레벨2가 없는 경우 레벨2_ID와 레벨2_라벨은 비워두세요.
4. **카테고리 ID**: 카테고리 ID는 정확히 입력해야 합니다 (대소문자 구분).

```csv
- 항목 1
- 항목 2"
```

